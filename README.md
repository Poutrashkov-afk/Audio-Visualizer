# Audio Visualizer
A real-time audio visualizer built with Python, Pygame, and PyAudio. It captures audio input from the microphone and generates dynamic visual effects based on the sound frequencies.
## Features
- Real-time audio capture and visualization
- Multiple visual effects (bars, circles, particles)
- Customizable settings (color, sensitivity, fade speed)
- Cross-platform compatibility (Windows, macOS, Linux)
## Installation
1. Clone the repository:
   ```bash
   git clone
    ```
2. Navigate to the project directory:
    ```bash
    cd audio-visualizer
    ```
3. Install the required dependencies:
    ```bash
    pip install -r requirements.txt
    ```
## Usage
Run the main script to start the visualizer:
```bash
python main.py
```
## Customization
You can customize the visualizer settings by editing the `settings.py` file. Adjust parameters such as color, sensitivity, and fade speed to create your desired visual effects.
